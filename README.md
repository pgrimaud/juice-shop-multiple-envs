# OWASP Juice Shop multiple envs
Run multiple OWASP Juice Shop containers on same host

## Start 5 containers

```sh
  ./run.sh 5
```

This will create 5 containers on ports : 
- 3001
- 3002
- 3003
- 3004
- 3005

## Expose Juice Shops

Just create vhosts on :
- Apache
- NGINX
- Other...
